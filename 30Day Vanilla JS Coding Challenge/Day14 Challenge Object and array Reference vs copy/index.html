Objects and Arrays - Reference VS Copy
In case of number and string variables , we copy them by value and not by reference
When arrays are copied , they are done so with reference
They can be copied with assignment operator
Array can be copied without referencing it with help of Slice,spread operator and Array.from- as they all make a complete new array and doesnt alter the original one
Objects are also copied and to do so without referencing it we use object.assign and also with JSON.parse(JSON.stringify())
